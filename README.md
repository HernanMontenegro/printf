# _Printf
In this project we are trying to create our own printf function from the standard library. 
**Handles**
 - %c - char format
 - %s - string format
 - % - percent
 - %d - decimal format
 - %i - int format
 - %b - binary format
 - %u - unsigned int format
 - %o - octal format
 - %x - lowercase hexadecimal format
 - %X - UPPERCASE hexadecimal format
 - %r - reverse string format
 - %R - rot13 format
